# Results Viewer

This is a local React dashboard designed to analyze and compare run data generated by the main Game agent. It visualizes the JSON logs emitted by the game natively in your browser, running completely local without requiring a backend for data processing.

## Features

- **Timeline View**: Visually track the sequence of encounters, HP/XP, decisions, skills unlocked, and tool usage across runs.
- **Answer Inspector**: A rich data table of all LLM inferences and choices made in the game. You can filter and drill down into the 'Internal Reasoning' thoughts generated by the model. 
- **Skill Tree**: Visualize which skills were unlocked and when.
- **Abilities Breakdown**: Aggregated usage tracking for player actions (Flee, Double Down, Reroll).
- **Self-Awareness**: Stat progression mapped over the sequence of the chambers.
- **Diff Comparison**: Side-by-side raw JSON comparison of finalized run files.

## Running the UI

1. Open your terminal and change directories into this folder.
   ```bash
   cd dungeon_of_self/backend/results_viewer
   ```

2. *(First Time Only)* Install the dependencies:
   ```bash
   npm install
   ```

3. Start the Vite development server:
   ```bash
   npm run dev
   ```

The dashboard will open automatically in your browser (usually `http://localhost:5173`). 

---

**Note**: When using the application, import runs by using the "Load Runs" function in the sidebar to load the ZIP files or raw folder selections containing valid JSON (`full_run.json`, `abilities.json`, etc.) from the backend logs.
